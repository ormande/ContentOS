# ContentOS

Sistema local para organizar ideias, vídeos, textos, versões de arquivos, publicações e biblioteca editorial.

## Como abrir

Depois de instalar Node.js, abra um terminal nesta pasta e rode:

`npm run dev`

Ou use o arquivo `start-localhost.bat`.

Ele abre o ContentOS em:

`http://localhost:4179`

## IA

A IA está prevista na arquitetura, mas desativada nesta primeira versão.
